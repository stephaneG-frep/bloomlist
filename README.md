# BloomList

BloomList est une application Flutter de gestion de taches premium:
- dashboard de progression
- gestion complete des taches (CRUD, sous-taches, priorites, tags, categories)
- projets, calendrier, kanban
- statistiques
- mode focus + minuteur Pomodoro
- suggestions selon l'energie
- theming clair/sombre personnalisable
- stockage local avec Hive

## Stack
- Flutter + Dart
- Riverpod
- GoRouter
- Hive

## Lancer
```bash
flutter pub get
flutter run
```
